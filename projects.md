### <a href="https://github.com/seandavidreed/simple-synth" target="_blank">Simple Synth</a>
This little program is a part of a larger effort to build a midi controller that takes input from hand gestures. I built this project as a way to practice C++ syntax and design patterns, and to gain a better understanding of how audio is represented digitally. I have much more to learn; this project only scratches the surface. I suspect I will modify it further in the future.

### <a href="https://github.com/seandavidreed/student-log" target="_blank">Student Log</a>
This console application was my first full, self-assigned programming project, and I have been using it for my job ever since I finished it! I completed the first version in June 2022, but I went on to optimize the codebase much further, creating versions 3 and 4, which are the ones present in the repository. It was designed to expedite the process of storing and fetching lesson notes for individual students.

Before I developed Student Log, I was using a spreadsheet to keep track of all my student information. This worked well, but I didn't like that the interface required so much interaction, copying and pasting, using the trackpad on my laptop. I wanted all the features I needed to be a mere keypress away. Though I could have accomplished this with the spreadsheet, I took the opportunity to hone my C Programming skills and build a system from the ground up. Moreover, by building it as a console application, I could eliminate the hefty overhead of spreadsheet software. This project confirmed my obsession with the C language and really catalyzed my programming journey as a whole.

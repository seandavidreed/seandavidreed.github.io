### <a href="https://github.com/seandavidreed/student-log" target="_blank">Student Log</a> ###
This console application was my first full programming project, and I have been using it for my job ever since I finished it! It was designed to expedite the process of storing and fetching lesson notes for individual students. Before I developed Student Log, I was using a spreadsheet to keep track of all my student information. This worked well, but I didn't like that the interface required so much interaction, copying and pasting, using the trackpad on my laptop. I wanted all the features I needed to be a mere keypress away. Though I could have accomplished this with the spreadsheet, I took the opportunity to hone my C Programming skills and build a system from the ground up. Moreover, by building it as a console application, I could eliminate the hefty overhead of spreadsheet software. This project confirmed my obsession with the C language and really catalyzed my programming journey as a whole.
